# Console-based Chatbot
This is a console-based chatbot project completed as part of my Python internship at CodeAlpha (Task 2).

# Features:
Interactive chatbot that responds to user inputs like greetings, jokes, and simple conversation.
Hndles case-insensitive inputs for better user experience.
Includes functions to structure the code cleanly and make it reusable.
Exit option using "bye" command.

# Concepts Used:
Loops (while)
Conditional Statements (if-elif-else)
String Handling (.strip(), .lower())
Functions (def)
input/output
